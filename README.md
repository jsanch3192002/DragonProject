# RGB Dragon — GitHub Pages

## Upload on iPhone
1. Create a new GitHub repository.
2. Upload all four files from this ZIP:
   - index.html
   - style.css
   - app.js
   - dragon.glb
3. Open the repository's **Settings**.
4. Open **Pages**.
5. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
6. Tap **Save**.
7. Wait about 1–3 minutes, then open the GitHub Pages link.

Keep all files in the repository root. Do not rename dragon.glb.
